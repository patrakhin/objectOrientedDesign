У Bird есть специфичные свойства и методы, такие как fly(), которые отличают птиц от других животных.

class Animal {
    // общие свойства и методы для всех животных
}

class Bird extends Animal {
    void fly() {
        // реализация полета для птиц
    }
}

Пример 2:
Класс Car ("автомобиль") является подклассом класса Vehicle ("транспортное средство").
У Car есть свои уникальные характеристики и методы, такие как drive(),
которые отличают автомобили от других видов транспорта.

class Vehicle {
    // общие свойства и методы для всех транспортных средств
}

class Car extends Vehicle {
    void drive() {
        // реализация движения для автомобилей
    }
}

Эти примеры демонстрируют отношение "является", где подкласс (например, Bird или Car)
явно наследует общие характеристики и поведение от суперкласса (например, Animal или Vehicle).
В этих случаях использование отношения "содержит" было бы менее естественным,
поскольку наследование подчеркивает естественную иерархию сущностей в рамках общей классификации.
