
Решение задания 19.

Примеры наследования:
Наследование подтипов (subtype inheritance):
Словесное описание: Рассмотрим класс Vehicle. Этот класс имеет подклассы Car ("автомобиль") и Bicycle .
Оба подкласса представляют конкретные подтипы транспортных средств.

class Vehicle {
    // общие свойства и методы для всех транспортных средств
}

class Car extends Vehicle {
    // дополнительные свойства и методы для автомобиля
}

class Bicycle extends Vehicle {
    // дополнительные свойства и методы для велосипеда
}


Наследование с ограничением (restriction inheritance):
Словесное описание: Рассмотрим класс Person ("человек"). Создадим подклассы Adult  и Child.
Ограничение будет заключаться в том, что у Child будет дополнительное ограничение на возраст, например, менее 18 лет.

class Person {
    // общие свойства и методы для всех людей
}

class Adult extends Person {
    // дополнительные свойства и методы для взрослого
}

class Child extends Person {
    // дополнительные свойства и методы для ребенка
    // возможно, дополнительное ограничение на возраст
}


Наследование с расширением (extension inheritance):
Словесное описание: Рассмотрим класс List ("список"). Создадим подклассы Stack и Queue.
При этом Stack расширяет базовый функционал List, добавляя операции push и pop,
а Queue также расширяет базовый функционал, предоставляя методы enqueue и dequeue.

class List {
    // общие свойства и методы для списка
}

class Stack extends List {
    // дополнительные свойства и методы для стека
    void push(Object item) {
        // реализация добавления элемента в стек
    }

    Object pop() {
        // реализация извлечения элемента из стека
    }
}

class Queue extends List {
    // дополнительные свойства и методы для очереди
    void enqueue(Object item) {
        // реализация добавления элемента в очередь
    }

    Object dequeue() {
        // реализация извлечения элемента из очереди
    }
}
